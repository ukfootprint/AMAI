# Deferred With Reason
*Ideas worth revisiting — but not yet. Every entry must explain why it's deferred, not just that it is.*

---

## Why This File Exists

Good ideas arrive at the wrong time. This file captures them without letting them become a dumping ground.

The name is intentional. "Backlog" is frictionless — any idea can slip in, and the AI will always offer to add to it. "Deferred with reason" requires a decision: *why isn't this right now?* That friction is a feature. If you can't articulate why something is deferred, it either should be a goal or should be dropped.

**Rule:** Every item requires a `Why not now` entry before it can be added.

## Review Cadence

This file is reviewed during **quarterly planning** (see `operations/rituals.md`). At each review:

1. Does each deferred item still have a valid reason for deferral?
2. Has anything become timely? → Move to `goals/goals.yaml`
3. Has anything lost its appeal? → Move to Dropped with a note

*An idea without a review date is an idea without a plan to revisit it.*
