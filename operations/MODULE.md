# Operations Module
*Load for: planning sessions, workflows, understanding how you work*

---

## What This Module Is For

This module captures how you work — your repeatable processes, tool stack, and recurring commitments. Load it when the AI needs to understand your working patterns or help you plan, schedule, or execute recurring tasks.

---

## Files In This Module

| File | Format | Load When |
|------|--------|-----------|
| `workflows.md` | Markdown | Executing a recurring process; reviewing a standard procedure |
| `tools.md` | Markdown | Recommending tools; understanding your current stack |
| `rituals.md` | Markdown | Weekly/monthly planning; reviewing commitments |

---

## AI Instructions

1. **Workflows are reference, not prescription.** Use documented workflows as defaults, but adapt them when the situation calls for it.
2. **Tool stack context.** When recommending workflows or integrations, check `tools.md` first — recommend approaches that fit the tools already in use.
3. **Rituals protect focus.** If something is listed in `rituals.md` as a committed practice, don't schedule over it or deprioritise it without flagging the conflict.
